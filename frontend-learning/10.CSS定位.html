CSS定位详解

掌握五种CSS定位方式：static、relative、absolute、fixed和sticky

静态定位 (Static)

静态定位是元素的默认定位方式，元素按照正常的文档流进行排列。

特点： 不受top、bottom、left、right属性影响，元素处于正常文档流中。
容器
Static
普通元素
普通元素
.static-box { position: static; /* 默认值，可省略 */ /* top、bottom、left、right属性无效 */ }

相对定位 (Relative)

相对定位的元素相对于其正常位置进行偏移，但原本所占的空间保留。

特点： 相对于自身原始位置偏移，不影响其他元素布局，原本空间保留。
容器
普通元素
Relative
普通元素
原始位置
.relative-box { position: relative; left: 120px; /* 相对于原始位置向右移动120px */ top: 40px; /* 相对于原始位置向下移动40px */ }

绝对定位 (Absolute)

绝对定位的元素相对于最近的已定位祖先元素进行定位，如果没有，则相对于文档主体。

特点： 脱离正常文档流，不占据空间，相对于最近的定位祖先元素定位。
容器 (position: relative)
普通元素
Absolute
普通元素
/* 容器样式 */ .position-container { position: relative; /* 为绝对定位元素提供参考 */ } .absolute-box { position: absolute; left: 50px; /* 相对于容器的左边缘 */ top: 50px; /* 相对于容器的上边缘 */ }

固定定位 (Fixed)

固定定位的元素相对于浏览器窗口进行定位，即使页面滚动，它也会停留在相同的位置。

特点： 脱离正常文档流，相对于浏览器视口定位，不随页面滚动而移动。
容器

固定定位元素相对于浏览器窗口定位，请查看页面右下角

.fixed-box { position: fixed; right: 30px; /* 相对于浏览器窗口右边缘 */ bottom: 30px; /* 相对于浏览器窗口下边缘 */ }

粘性定位 (Sticky)

粘性定位是相对定位和固定定位的混合。元素在跨越特定阈值前为相对定位，之后为固定定位。

特点： 基于用户的滚动位置在相对定位和固定定位之间切换。

定位属性总结

定位方式 参考点 是否脱离文档流 特点
static 正常文档流 否 默认值，不受定位属性影响
relative 自身原始位置 否 偏移不影响其他元素，原本空间保留
absolute 最近定位祖先元素 是 脱离文档流，不占空间
fixed 浏览器视口 是 不随页面滚动，始终在视口中固定位置
sticky 视口和父元素 否/是 滚动到阈值前相对定位，之后固定定位
Fixed 固定定位元素
