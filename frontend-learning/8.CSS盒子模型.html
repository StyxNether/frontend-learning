CSS盒子模型详解

注意： 此页面已设置 box-sizing: border-box，这意味着元素的宽度和高度包括了内容、内边距和边框，但不包括外边距。

基本盒子模型示例

总尺寸: 300px × 150px
内容区域
/* CSS代码 */ .demo { width: 300px; /* 内容宽度 */ height: 150px; /* 内容高度 */ padding: 30px; /* 内边距 */ border: 10px solid #e74c3c; /* 边框 */ margin: 20px; /* 外边距 */ background-color: #f1c40f; /* 背景色 */ }

边框样式示例

边框演示
/* CSS边框代码 */ .border-demo { /* 复合边框 - 四个方向不同 */ border-style: solid dashed dotted double; border-width: 10px 5px 15px 20px; /* 上 右 下 左 */ border-color: blue green red yellow; /* 单独设置左边框 - 覆盖复合设置 */ border-left: dotted red 5px; }

盒子模型可视化

外边距 (Margin)
边框 (Border)
内边距 (Padding)
内容 (Content) 实际内容

盒子模型属性详解

属性 描述 示例值
width / height 设置内容区域的宽度和高度 300px, 50%, auto
padding 内容与边框之间的空间 20px, 10px 20px, 5px 10px 15px 20px
border 围绕内容和内边距的边框 2px solid #000
margin 元素外部的空间，与其他元素分开 10px, 10px auto, 5px 10px 15px 20px
box-sizing 定义如何计算元素的宽度和高度 content-box, border-box
提示： 使用 box-sizing: border-box 可以更直观地控制元素尺寸，因为它将边框和内边距包含在宽度和高度计算中。
